Video of the website running is in the brightspace submission.

Run Instructions:
- Clone https://github.com/JoshLopez26/498-Webserver.git into your server and pull
  If the pull was successful, there should be a new directory called '498-Webserver'
- cd into '498-Webserver' then run the following commands
- docker compose build
- docker compose up -d
- In your browser, open http://159.65.47.11/
- To close, run docker compose down
